🚀 COMO COMEÇAR
═══════════════════════════════════════════════════════════════════════════════

Opção 1: Visualizar Direto
   1. Abra a pasta do projeto
   2. Clique 2x em "index.html"
   3. A página abrirá no navegador

Opção 2: Servidor Local (Recomendado)
   # Com Python 3
   python -m http.server 8000
   # Acesse: http://localhost:8000

Opção 3: Live Server (VS Code)
   1. Clique direito em "index.html"
   2. Selecione "Open with Live Server"

Opção 4: Outros Servidores
   # Node.js
   npx http-server
   
   # Ruby
   ruby -run -ehttpd . -p8000


🔧 PERSONALIZAÇÕES COMUNS
═══════════════════════════════════════════════════════════════════════════════

Mudar Cor Vermelha
   └─ Abra: style.css
   └─ Procure: --color-primary-red: #d32f2f;
   └─ Mude o código: #d32f2f → seu-código

Editar Conteúdo
   └─ Abra: index.html em editor de texto
   └─ Procure o texto
   └─ Edite livremente

Mudar Tipografia
   └─ Abra: style.css (topo)
   └─ Procure: @import url(...fonts.googleapis.com...)
   └─ Mude a fonte Google Fonts

Adicionar Seção
   └─ Abra: CSS_SNIPPETS.md
   └─ Use o template pronto
   └─ Copie e cole no HTML

💡 DICAS IMPORTANTES
═══════════════════════════════════════════════════════════════════════════════

→ Use as variáveis CSS (--color-primary-red)
  NUNCA hardcode cores (#d32f2f)

→ Teste em mobile SEMPRE após editar
  Responsividade é crítica

→ Mantenha hierarquia de heading (h1→h2→h3)
  Importante para SEO e acessibilidade
