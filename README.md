# ACIERJ - Associação dos Cuidadores do Estado do Rio de Janeiro

## 🌐 Site Institucional

Bem-vindo ao repositório do site da **ACIERJ** - uma organização da sociedade civil sem fins lucrativos dedicada à defesa dos direitos dos cuidadores e à promoção de justiça social.

---

## 📋 Sobre a ACIERJ

A ACIERJ é uma organização que atua na defesa dos direitos de cuidadores e está alinhada com pautas sociais fundamentais:

- ✊ **Direitos da Pessoa Idosa**
- ♿ **Inclusão de Pessoa com Deficiência**
- 🏘️ **População em Situação de Rua**
- 🧠 **Luta Antimanicomial**
- 💜 **Inclusão LGBTQIA+**
- 👩🏾 **Movimento de Mulheres Negras**
- ⚖️ **Justiça Social e Direitos Humanos**

---

## 🛠️ Tecnologias Utilizadas

- **HTML 5** - Semântico e acessível
- **CSS 3** - Modular com variáveis CSS
- **Google Fonts** - Tipografia: Unna (serif) e Montserrat (sans-serif)
- **Sem JavaScript** - Funcionalidade pura com HTML/CSS

## 🚀 Como Usar

### Visualizar Localmente

```bash
# 1. Clone o repositório
git clone https://github.com/Famoveover/acierj-cuidadores.git

# 2. Navegue até a pasta
cd acierj-cuidadores

# 3. Abra o arquivo no navegador
# Opção 1: Duplo clique em index.html
# Opção 2: Abra no navegador (Ctrl+O ou Cmd+O)
# Opção 3: Use um servidor (Live Server no VS Code)
```

### Servidor Local (Recomendado)

```bash
# Com Python 3
python -m http.server 8000

# Com Python 2
python -m SimpleHTTPServer 8000

# Com Node.js (http-server)
npx http-server

# Com VS Code Live Server
# Clique direito em index.html > "Open with Live Server"
```

Então acesse: `http://localhost:8000`
